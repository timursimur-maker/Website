Wichtige Struktur:
Start
├── Serien
│   ├── Türen & Glas

│   ├── Durchblicke
│   ├── Formen
│   └── Ligurien

└── Kontakt

Upload:
1. ZIP auf dem Handy entpacken.
2. Alle enthaltenen Dateien direkt in das Hauptverzeichnis des GitHub-Repositories hochladen.
3. Vorhandene gleichnamige Dateien ersetzen.
4. Commit changes bestätigen.
5. GitHub Pages anschließend neu laden.

Bereits im Repository vorhandene Bilddateien, die von den Serien verwendet werden,
müssen nicht erneut hochgeladen werden.
